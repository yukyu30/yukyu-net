OpenAI APIを活用したクイズ生成Webアプリケーションを開発しました。

![Quiz Bite](./quiz-bite.png)

## 概要

Quiz BiteはNext.js、Rails、LangChain、OpenAI APIを組み合わせて構築したクイズ生成アプリケーションです。AIを活用して様々なジャンルのクイズを自動生成します。

## 技術スタック

- Next.js
- Rails
- LangChain
- OpenAI API
