OpenAI APIを活用したRust製Discord botを開発しました。
![ai-discord-bot](/posts/2023-03-17-ai-discord-bot/bot.png)

## 概要

RustとOpenAI APIを使用して構築したDiscord botです。Shuttleへデプロイすることで、誰でも利用可能なAIアシスタントとして機能します。

## 技術スタック

- Rust
- OpenAI API
- Discord API
- Shuttle (デプロイプラットフォーム)
